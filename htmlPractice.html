HTML 구성요소

    

    요소(element): 시작태그와 종료태그로 이루어진 HTML 문서의 구성 요소이다.
    속성(attribute): 요소에 대한 추가적인 정보(색, 크기, 링크등)를 제공한다.
    *tip* 텍스트는 body태그 안에 특별한 태그 없이 출력 가능하다. 하지만 단락을 사용하지 않으면 전체가 연결 되어 한줄로 표시된다.
    br태그: 줄바꿈 = enter (HTML코드에서 줄을 바꾼다고 웹페이지에서 적용되진 않는다.)
    hr태그: 가로줄을 긋는다.
    pre태그: 프로그래머가 입력한 그대로(엔터도 적용) 출력된다. P.S.이설명을 적으면서 사용하고 있는 태그

    
텍스트 서식

    b태그: 볼드
    i태그: 이탤릭체
    strong태그: 텍스트를 강하게 표시한다.
    em태그: 텍스트를 강조한다.
    code태그: 코드임을 나타냄
    sup태그: 글자위첨자
    sub태그: 글자아래첨자
